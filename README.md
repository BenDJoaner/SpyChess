# 间谍象棋 — Spy Chess

一款 9×9 双人对战/单人 AI 棋盘游戏，核心特色是**间谍机制**——双方在开局时秘密埋设内鬼，对局中可突然揭露令对手措手不及。

---

## 项目结构

```
SpyChess/
├── game.py              # 主入口，运行 python game.py 启动游戏
├── engine.py            # 核心逻辑层（棋盘、棋子、走法、晋升、间谍）
├── renderer.py          # pygame 渲染层（棋盘、棋子、UI 面板）
├── ui.py                # 交互状态机（鼠标/键盘事件 → engine 调用）
├── ai.py                # AI 模块（Alpha-Beta 搜索 + 估值函数 + 战术系统）
├── move_log.py          # 棋谱记录（实时追加、回溯、导出）
├── replay.py            # 录像回放控制器
├── save_load.py         # 存档与读档（JSON 格式）
├── gen_icon.py          # 图标生成工具
├── SpyChess.spec         # PyInstaller 打包配置
├── resource/
│   ├── icon.ico         # Windows 程序图标
│   └── icon.png         # 透明 PNG 图标
└── saves/
    └── replay_*.json    # 录像存档
```

## 技术特点

### 分层架构（逻辑与表现分离）

```
┌─────────────────────────────────────────────┐
│  ui.py     — 交互状态机，鼠标/键盘事件处理   │
│            调用 engine，不直接操作棋盘数据    │
├─────────────────────────────────────────────┤
│  engine.py — 纯逻辑层，无 UI/渲染依赖        │
│            棋盘、棋子、走法、晋升、间谍规则   │
├─────────────────────────────────────────────┤
│  renderer.py — pygame 渲染，只读 engine 数据│
│              负责棋盘/棋子/面板/高亮绘制     │
├─────────────────────────────────────────────┤
│  ai.py     — AI 模块，调用 engine 走棋      │
│            支持接入任意外部 AI（实现接口即可） │
└─────────────────────────────────────────────┘
```

### 棋盘与坐标系统

- **9×9 棋盘**，坐标系 `(x, y)`，`board[y][x]`（y 是行，x 是列）
- 红方在左（`x<4`），蓝方在右（`x>4`），中列为中立区（`x==4`）
- 红方前进 `y-1`，蓝方前进 `y+1`；**渲染时 y 轴反向**（y=8 画在顶部）
- 两格**堡垒格** `(2,4)` `(6,4)`：进入后本回合无法再次行动，下回合必须先移出

### 棋子体系（11 种，含 Tier 晋升）

| Tier | 类型 | 说明 |
|------|------|------|
| T0 | 士兵 Soldier | 直线前进，斜前方吃子；走到底线升大臣，吃子升军官 |
| T1 | 大臣 Minister / 军官 Officer | 大臣可被选为间谍；吃子后按晋升表升级 |
| T2 | 骑士/刺客/铁卫/总督/教主 | 各有独特移动机制（跳跃/炮击/远程/翻越） |
| T3 | 亲信 Loyalist / 御史 Censor | 亲信活动范围受限；御史无线射程 |
| T4 | 领主 Lord | 胜负核心，被吃即游戏结束；拥有"直杀"特殊能力 |

### 间谍机制（核心创新）

**选间谍阶段**（4 步，回合制）：
1. 红方选择 1 枚对方**大臣**为间谍（知具体位置）
2. 蓝方选择 1 枚对方**大臣**为间谍
3. 红方点"随机士兵"→系统从蓝方士兵中随机抽 **2 枚**为间谍（不知哪枚）
4. 蓝方同理

**揭露阶段**：己方回合可揭露所有间谍，棋子**阵营立即倒戈**，本回合结束。
> 保密规则：除指定大臣外，任何人（甚至己方）事后无法从棋盘判断谁是士兵间谍。

### AI 系统（ai.py）

- **搜索算法**：Alpha-Beta 剪枝 + 迭代加深
- **Human-Like 模式**：评分相近的走法随机选取，避免每步必选最优的"机器感"
- **战术系统**：8 种预设战术（偷鸡、稳健、攻势等）+ 开局/中盘/残局动态切换
- **4 档难度**：Easy / Normal / Hard / Hell
- **信息对称原则**：AI 对士兵间谍的认知与人类一致（不知道具体 ID）

### 其他功能

- **棋谱记录**（move_log.py）：实时追加、点击回溯、导出文本
- **录像回放**（replay.py / save_load.py）：JSON 存档，可随时重放
- **热键支持**：S=揭露间谍 / U=悔棋 / M=显示可动棋 / H=高亮最后一步 / Esc=暂停

## 玩法简介

### 开局

1. 选择红/蓝方（人类或 AI）
2. **选间谍阶段**（4 步）：每方各指定 1 枚大臣 + 随机 2 枚士兵为隐藏间谍
3. 进入正式对局

### 对局操作

| 操作 | 说明 |
|------|------|
| 点击己方棋子 | 选中，显示绿色可移动位 / 红色可吃子位 |
| 点击目标位置 | 移动或吃子 |
| 点击对方棋子（非吃子位） | 忽略 |
| 揭露间谍 | 消耗本回合，将所有己方间谍倒戈为对方阵营 |
| 悔棋 | 撤销上一步（含对方回合）|

### 胜利条件

吃掉对方**领主（Lord）**即获胜。

> 注意：领主有"直杀"能力——若两颗领主在同一直线/斜线上且之间无子，一方可隔着中立区直接吃掉对方领主。

### 特殊机制一览

- **堡垒格**：进入后本回合强制结束，下回合必须先移出才能操作其他棋子
- **晋升**：士兵/大臣/军官吃子后根据被吃类型自动晋升为更强棋子
- **间谍揭露**：最关键的翻盘手段，需在对局中寻找时机主动揭露
